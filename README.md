# jest-test-component


### Installation

```sh
$ npm install -g @vue/cli
```

```sh
$ vue create vue-test-component-with-jest
```

```sh
$ vue create vue-test-component-with-jest
```

```sh
$ npm install @vue/cli-plugin-unit-jest @vue/test-utils
```

### Create jest.config.js

```sh
    module.exports = {
        preset: '@vue/cli-plugin-unit-jest'
    }
```


### Run Test File

```sh
$ npm test
```
